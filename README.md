# 🔐 PassOP - React Password Manager

A simple password manager built with **Vite**, **React**, and **Tailwind CSS**.

## ⚙️ Stack

- ⚛️ React + Vite  
- 🎨 Tailwind CSS  
- 💾 LocalStorage for data persistence

## 🚀 Setup

```bash
git clone https://github.com/pritamdev-bit/PassOP.git
cd password-manager
npm install
npm run dev
